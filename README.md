Pine City Zoo is a multi-page website built using HTML and CSS, designed to provide an engaging and informative online experience for zoo visitors. The site includes an Animals page showcasing various species, a Map page to help navigate the zoo grounds, a Places page highlighting key attractions and facilities, a Weather page to keep visitors updated on current conditions, and a Feedback page for collecting visitor opinions and suggestions. This project demonstrates foundational web development skills and offers a user-friendly layout focused on accessibility and ease of navigation.
